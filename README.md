# Smart Waste Management System - SDG 12: Responsible Consumption and Production

## Project Description
A Windows Forms application for tracking household waste management to promote responsible consumption and waste reduction.

## Installation
1. Clone the repository
2. Open SmartWaste.UI.sln in Visual Studio
3. Restore NuGet packages
4. Build and run

## SDG Justification
This project supports UN SDG 12 by helping households track and manage their waste production, promoting awareness and reduction of waste.
